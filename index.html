How do I write things in Swift?test



Classes

class className {
   statements
}


Protocols

protocol protocolName {
   protocolFunctions
}


Enums

enum enumName {
   case memberValue, anotherMemberValue
}


Structs

struct structName {
   statements
}


Mutating Properties of a Structure

struct structName {
   mutating func functionName {
      statements
   }
}


Functions

func functionName(parameters) -> returnType {
   statements
}


Class-Level Functions

class func functionName(parameters) -> returnType {
   statements
}


Extensions (Categories)

extension classToExtend: optionalProtocol {
   statements
}


In-out parameters

func functionName(inout parameterName: parameterType) {
   statements
}


Subscripting

struct structToSubscript {
   let constantToSubscript = value

   subscript( parameterName: parameterType ) -> returnType {
      statements
   }
}


Unowned Reference Usage

var closure = { [unowned self] in
   self.property
}


Closures

{ (parameters) -> returnType in statements }

or

{ (parameters) -> returnType in
   statements
   return result
}


Generics

func functionName<T>(parameterName: T) -> returnType {
   statements
}


Type-Casting

expression as type
expression as? type


Runtime Type-Checking

expression is type


String interpolation

println("Swift makes me feel \(object) inside")


Selectors

#selector(SomeClass.someMethod(_:)) #selector(getter: SomeClass.someProperty) #selector(setter: SomeClass.someProperty) #selector(SomeClass.doSomething(_:) as (SomeClass) -> (String) -> Void)


Attributes Reference

@assignment
Apply this attribute to functions that overload a compound assignment operator. Functions that overload a compound assignment operator must mark their initial input parameter as inout.

@class_protocol
Apply this to a protocol to indicate that the protocol can be adopted by class types only.

@exported
Apply this to an import statement. Any modules that import this module will have access to its exports.

@final
Apply this to a class to indicate that it cannot be further subclassed.

@lazy
A lazy stored property is a property whose initial value is not calculated until the first time it is used.

@NSCopying
Behaves similarly to the objective-c copy property attribute

@NSManaged
Indicates that this property is managed by Core Data at runtime

@objc
Apply this attribute to any declaration that can be represented in Objective-C. Protocols marked with the @objc can't inherit from protocols that aren't.

@optional / @required
Function similarly to objective-c, marking methods as optional or required within a protocol.

@auto_closure
Delay the evaluation of an expression by wrapping it in a closure with no arguments.

@noreturn
Indicate that the type or function doesn't return to its caller.
